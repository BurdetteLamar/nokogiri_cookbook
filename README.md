# Nokogiri Cookbook

## Nokogiri

Gem `nokogiri` is the most widely used HTML/XML parsing library for Ruby.
It provides a Ruby-friendly API
for reading, querying, editing, and writing HTML and XML documents,
with support for CSS selectors and XPath.

It is built on top of native parsers such as libxml2,
making it both fast and standards-compliant.

To install `nokogiri`, see [Installing  Nokogiri][installing].
